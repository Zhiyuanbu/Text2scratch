Privacy Policy

Last updated: February 19, 2026

1. Local Processing

text2scratch converts scripts to .sb3 in your browser. Your script content is not sent to a backend server by this app.

2. Files and Downloads

Generated .sb3 and .t2sh files are created locally and downloaded directly to your device.

3. Third-Party Services

This site loads frontend libraries from CDNs, including JSZip (for file packaging) and Monaco Editor (for the code editor). Your browser may contact these services to fetch those assets.

4. Imported Files

When you import .sb3 or .t2sh files, conversion happens in-browser. Imported project content is not uploaded by this app.

5. Hosting Logs

If hosted on GitHub Pages or another host, the host may collect standard HTTP logs (IP address, user-agent, timestamps) according to the host's own policy.

6. Contact

If you deploy this project publicly, add your own contact details and any required regional privacy disclosures.
