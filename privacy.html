Privacy Policy

Last updated: February 19, 2026

1. Local Processing

text2scratch converts scripts to .sb3 in your browser.

If you use cloud features (account, save, share), project text is stored in Supabase under your account and may be read by others only when you generate a public share link.

2. Files and Downloads

Generated .sb3 and .t2sh files are created locally and downloaded directly to your device.

3. Third-Party Services

This site loads frontend libraries from CDNs, including JSZip (for file packaging), Monaco Editor (for the code editor), Supabase (authentication and project storage), and Datadog Browser RUM (for monitoring).

Datadog RUM may collect performance metrics, JavaScript errors, page views, and sampled session replay data. Input fields are masked by default using mask-user-input.

4. Imported Files

When you import .sb3 or .t2sh files, conversion happens in-browser. Imported project content is not uploaded by this app.

5. Hosting Logs

If hosted on GitHub Pages or another host, the host may collect standard HTTP logs (IP address, user-agent, timestamps) according to the host's own policy.

6. Contact

If you deploy this project publicly, add your own contact details and any required regional privacy disclosures.
